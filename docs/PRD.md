# SpeakWise Studio PRD

## 1. 产品定位

SpeakWise Studio 是一个帮助用户在真实社交场景中训练表达能力的 AI 陪练产品。

用户不是来学“理论”，而是来练“我现在该怎么回”。

## 2. 目标用户

- 刚进入职场，希望说话更得体的年轻人
- 容易在家庭、亲戚、社交关系里被问住的人
- 希望提升沟通边界感、表达清晰度的人

## 3. 核心问题

用户常见痛点：

- 当下想不到合适的回答
- 回答太冲、太软、太绕
- 事后复盘才发现自己说错了
- 不知道“更好”的表达长什么样

## 4. MVP 范围

### 4.1 首页

- 产品定位文案
- 场景分类入口
- 今日金句
- 表达原则卡片

### 4.2 练习页

- 展示场景说明和对方发言
- 记录用户输入内容
- 记录用户思考时间
- 提供语音模式入口

### 4.3 评分报告页

- 总分
- 五维评分
- 优点
- 风险点
- 更优回答
- 进阶表达

## 5. 场景分类

- 职场沟通
- 亲戚问话
- 社交边界
- 亲密关系

## 6. 评分维度

- 分寸感
- 情绪稳定
- 表达清晰
- 边界感
- 得体度

附加客观指标：

- 思考时长
- 回答长度

## 7. 成功标准

- 用户能在 3 分钟内完成一轮练习
- 每次练习都能拿到具体、可执行的建议
- 用户愿意重复练习多个场景

## 8. 第二阶段

- 语音录制与转写
- 用户账户体系
- 历史记录和成长报告
- 排行榜 / 连续打卡
- 会员高级场景
