# 小程序审核说明

## 产品简介

`eqora` 是一款表达训练类小程序，帮助用户在职场、家庭、社交、关系等常见场景中进行回应练习。用户可以通过文字或语音作答，系统会生成表达点评，并将训练结果沉淀到个人成长档案中。

## 核心功能

1. 首页查看今日推荐训练和训练重点
2. 进入练习场选择具体场景
3. 在训练页通过文字或语音完成作答
4. 提交后生成点评报告和优化建议
5. 在“我的”页查看成长记录、平均分和最近训练
6. 在“广场”页查看精选表达内容和训练提示

## 审核体验路径

建议审核员按以下路径体验：

1. 进入首页
2. 点击底部导航“练习场”
3. 选择任意场景进入训练页
4. 可直接输入文字作答，或点击“开始录音”体验语音转写
5. 点击“提交并生成点评”
6. 查看报告页中的评分、优点、风险和优化话术
7. 返回“我的”页查看训练记录是否已保存

## 登录与账号说明

- 小程序使用微信登录，无需额外注册测试账号
- 首次进入“我的”页后，点击“微信登录”即可完成登录
- 不存在邀请码、付费墙或外部跳转登录流程

## 权限说明

- 麦克风权限：仅在用户主动点击“开始录音”时申请，用于语音转写
- 即使不授权麦克风，也可以使用文字输入完整体验主流程

## 审核补充说明

- 训练主链路为：首页 -> 练习场 -> 训练页 -> 报告页 -> 我的
- 所有训练记录仅用于用户个人查看和持续练习
- 语音内容仅用于转写和训练反馈，不会向其他用户展示
- 如审核时希望快速验证主流程，推荐优先使用文字输入，体验更稳定
